3:T592,<h1>Balanced Parentheses with Minimum Edits</h1>
<p>This problem was asked by Facebook.</p>
<p>Given a string of parentheses, find the balanced string that can be produced from it using the minimum number of insertions and deletions. If there are multiple solutions, return any of them. </p>
<p>For example, given <code>&quot;(()&quot;</code>, you could return <code>&quot;(())&quot;</code>. Given <code>&quot;))()(&quot;</code>, you could return <code>&quot;()()()()&quot;</code>.</p>
<h2>How the Solution Works</h2>
<p>We can construct a balanced string with the minimum number of edits by tracking unmatched open parentheses:</p>
<ul>
<li>We traverse the string from left to right.</li>
<li>If a closing parenthesis <code>)</code> is encountered and there is no unmatched open parenthesis <code>(</code>, we insert a <code>(</code> directly before it.</li>
<li>If there is an unmatched <code>(</code>, we match it with the <code>)</code> and decrement our unmatched open counter.</li>
<li>At the end of the string, if any unmatched open parentheses <code>(</code> remain, we append the corresponding number of closing parentheses <code>)</code> to the end.</li>
</ul>
<h2>Complexity</h2>
<ul>
<li><strong>Time Complexity:</strong> $O(N)$ where $N$ is the length of the string, as we process each character exactly once.</li>
<li><strong>Space Complexity:</strong> $O(N)$ to build and return the balanced string.</li>
</ul>
